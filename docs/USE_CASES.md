# Use Cases

## Primary Use Cases

### 1. User Registration and Authentication
**Actor**: New Customer
**Goal**: Create account and access banking services

**Steps**:
1. Customer provides personal information
2. System validates information
3. Account is created with initial credentials
4. Customer receives confirmation

### 2. Account Management
**Actor**: Bank Customer
**Goal**: Manage bank accounts

**Steps**:
1. Customer logs into system
2. Views account dashboard
3. Can create new accounts
4. Update account information
5. View account history

### 3. Fund Transfer
**Actor**: Account Holder
**Goal**: Transfer money between accounts

**Steps**:
1. Select source and destination accounts
2. Enter transfer amount
3. System validates sufficient funds
4. Transaction is processed
5. Both accounts are updated
6. Confirmation is sent

### 4. Transaction History
**Actor**: Account Holder
**Goal**: View transaction history

**Steps**:
1. Access account dashboard
2. Select date range
3. Filter by transaction type
4. Export transaction data

### 5. Administrative Oversight
**Actor**: Bank Administrator
**Goal**: Monitor and manage system

**Steps**:
1. Access admin dashboard
2. View system metrics
3. Manage user accounts
4. Review transactions
5. Generate reports
